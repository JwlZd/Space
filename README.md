# The Great Space Timeline

The Great Space Timeline is a simple educational website about space exploration.  
It includes a timeline of important space events, a page about famous astronauts, and an about page explaining the project.

## Features

- A small timeline of major space milestones  
- Information about well-known astronauts  

## Built With
- HTML    

## Pages

- Timeline (Home Page)  
- Famous Astronauts  
- About  

## How to Run

1. Clone the repository:
2. Open the project folder.
3. Open `index.html` in your browser.

## Purpose
This project was created for Hack Club to practice building and organizing a multi-page website using basic web development skills.
